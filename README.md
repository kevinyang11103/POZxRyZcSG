# 项目简介 python1290

本项目是一个基于 Flask 和 Vue.js 的博客论坛系统，支持文章和消息通知功能。

## 技术栈

- 后端框架：Flask
- 前端框架：Vue.js
- 编程语言：Python 3.7/3.8
- 数据库：MySQL 5.7 或更高版本
- 数据库工具：Navicat 11（推荐）
- 开发环境：PyCharm、VS Code

## 功能模块

- 用户注册与登录
- 文章发布与浏览
- 消息通知
- 评论与互动

## 系统角色

- 普通用户：注册、登录、发布文章、评论、查看通知
- 管理员：用户管理、文章管理、评论管理

## 运行环境

- Python 3.7/3.8
- Flask
- Vue.js
- MySQL 5.7 或更高版本

## 部署步骤

1. 安装 Python 3.7/3.8
2. 安装 Flask、Vue.js 和 MySQL
3. 克隆项目代码
4. 创建数据库并导入数据
5. 配置项目环境
6. 运行项目

## 目录结构

```
project/
│
├── app/                # Flask 应用目录
│   ├── models/         # 数据库模型
│   ├── views/          # 视图处理
│   └── static/         # 静态文件目录
│
├── frontend/           # Vue.js 前端项目目录
│   ├── src/            # Vue.js 源码
│   └── public/         # 公共文件
│
└── migrations/         # 数据库迁移文件
```

## 核心亮点

- 前后端分离，便于维护和扩展
- 采用 Flask 框架，开发效率高
- 使用 Vue.js，实现单页面应用（SPA）效果
- 适应多种数据库版本，降低环境限制
